# Playing around with go
![example workflow](https://github.com/alainrk/gostuff/.github/workflows/go.yml/badge.svg)


## Mod init
```
go mod init github.com/alainrk/gostuff
```

## Build everything recursively
```
go build ./...
```

## Test everything recursively
```
go test ./...
```
